Estrutura Básica

A estrutura básica do HTML é composta pelos elementos !DOCTYPE html, no qual declaramos que o documento é um html, as tags html, head e body. O elemento html representa a raiz do documento html, e todos os outros elementos devem ser descendentes dele.

No head, colocamos as informações que o browser irá interpretar a respeito daquele documento em si, o que chamamos de metadados. Já no body, temos as informações e elementos que de fato irão aparecer na tela para o usuário.

Tags HTML

Existem inúmeras tags que podem ser usadas num documento HTML, algumas delas semânticas e outras não. Vamos citar aqui alguns exemplos de tags:

  • Tags <h1> a <h6>
  • Tag <p>
  • Tag <a>
  • Tag <hr>
  • Tags <ul>, <ol> e <li>

Atributos das Tags

Quando falamos de tags, logo aprendemos sobre os atributos. Atributos são como características que colocamos nas tags de forma a controlar o seu comportamento. Existem atributos globais, que podem ser aplicados a qualquer tag HTML e também existem os atributos específicos, que funcionam apenas para determinadas tags.

Alguns atributos globais que podemos citar: class, id, style.

Atributos específicos: href, target, width.

Textos

Nos documentos HTML, é muito comum haver textos, como em blogs e sites de notícias. Para estruturar bem o nosso texto, algumas das tags principais são as tags <h1> a <h6>, que funcionam da forma abaixo:

Em geral, devemos usar apenas um <h1> por página, indicando o título da mesma, e estruturar o restante do texto de acordo com as demais tags <h2> a <h6>.

As vezes, para números ou algum tipo de expressão podem ser usadas as tags <sub> e <sup>.

Veja como fica a fórmula química do ácido sulfúrico, se utiliza a tag <sub>:

H2SO4

Veja como fica o número 2 elevado a 3, utilizando a tag <sup>:

23

Uma observação: A tag font era usada até o HTML4, mas foi descontinuada no HTML5, pois hoje em dia é mais útil classificar fontes e estilos através do CSS.

Listas Ordenadas e Não-Ordenadas

Outro elemento muito importante em uma página são as listas, que podem ser do formato ordenado ou não ordenado. No formato ordenado, será indicada a ordem dos itens da lista, enquanto que no não ordenado, não há indicação de ordem.

As listas são muito usadas para fazer o menu das páginas web.

Links

Por último mas não menos importante temos a tag a, que faz referência a âncora. Ela serve para indicar links no documento HTML, seja para um link na mesma página, dentro do mesmo site ou links externos.

Utilizamos o atributo href para indicar o link, da seguinte forma:

<a href="https://www.google.com.br/">Clique para visitar o Google.</a>

Quando utilizada, o link ficará clicável, assim: Clique para visitar o Google.
